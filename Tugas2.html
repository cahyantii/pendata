Penjelasan Outlier Deteksi

Deteksi Outlier dengan K-Nearest Neighbors (KNN) dalam Data Understanding

  1. Apa itu outlier Deteksi?

Outlier detection adalah proses identifikasi data atau pengamatan yang memiliki nilai yang sangat berbeda (menyimpang) dari mayoritas data lainnya dalam sebuah dataset. Outlier ini bisa menunjukkan kesalahan pengukuran, variabilitas yang ekstrem, atau kejadian yang tidak biasa.

Penjelasan Kode

Kode ini mendeteksi outlier dalam dataset bunga iris dengan menggunakan jarak Euclidean terhadap titik referensi. Outlier ditentukan berdasarkan threshold persentil 93.5%, dan hasilnya divisualisasikan menggunakan scatter plot.

Langkah-Langkah Kode:

  1. Mengambil Data dari PostgreSQL dan MySQL

  • get_pg_data(): Mengambil data sepal (panjang & lebar sepal) dari PostgreSQL.

  • get_mysql_data(): Mengambil data petal (panjang & lebar petal) dari MySQL.

  • Data ini diambil menggunakan library psycopg2 dan pymysql, lalu dikonversi menjadi pandas DataFrame.

  1. Menggabungkan Data Berdasarkan ID dan Class

  • Data dari PostgreSQL dan MySQL digabungkan dengan pd.merge() menggunakan kolom “id” dan “class”.

  • Hanya data yang memiliki pasangan di kedua database yang dipertahankan (how=”inner”).

  1. Menghitung Jarak Euclidean

  • Fitur numerik yang digunakan:

  • petal_length, petal_width, sepal_length, sepal_width

  • Titik referensi yang dipilih adalah baris terakhir dari dataset.

  • Fungsi compute_distances() menghitung jarak Euclidean antara setiap titik dengan titik referensi.

  1. Menentukan Outlier

  • Threshold ditentukan berdasarkan persentil 93.5% dari nilai jarak yang dihitung.

  • Titik yang memiliki jarak lebih besar dari threshold dikategorikan sebagai outlier.

  1. Menampilkan Hasil

  • Data ditampilkan dengan kolom tambahan:

  • “distance” (jarak Euclidean ke referensi).

  • “outlier” (True jika termasuk outlier, False jika bukan).

  • Hasil divisualisasikan menggunakan scatter plot:

  • Warna biru untuk data normal, merah untuk outlier.

  • Scatter plot dibuat untuk sepal dan petal dengan Seaborn.

Kode ini mendeteksi outlier berdasarkan jarak Euclidean terhadap titik referensi terakhir di dataset. Data yang memiliki jarak terbesar (di atas persentil 93.5%) dianggap sebagai outlier dan divisualisasikan dalam bentuk scatter plot.

# Ambil data dari kedua database
df_postgresql = get_pg_data()
df_mysql = get_mysql_data()

# Gabungkan berdasarkan kolom 'id' dan 'Class'
df_merged = pd.merge(df_mysql, df_postgresql, on=["id", "class"], how="inner")

# Ambil data fitur numerik
feature_columns = ["petal_length", "petal_width", "sepal_length", "sepal_width"]
data_values = df_merged[feature_columns].values

# Ambil referensi dari baris terakhir (baris ke-152 jika dihitung dari 1, atau index -1)
reference_point = data_values[-1]

def compute_distances(data, reference):
    return np.array([euclidean(row, reference) for row in data])

# Hitung jarak Euclidean dari setiap baris ke referensi
df_merged["distance"] = compute_distances(data_values, reference_point)

# Tentukan threshold outlier berdasarkan persentil ke-95 (bisa disesuaikan)
threshold = np.percentile(df_merged["distance"], 93.5)
df_merged["outlier"] = df_merged["distance"] > threshold

# Cetak hasil data dengan outlier
print(df_merged.to_string(index=False))

# Visualisasi scatter plot dengan warna berdasarkan outlier
plt.figure(figsize=(8, 6))

    hue=df_merged["outlier"], palette={False: "blue", True: "red"}
)
plt.title("Outlier Detection (Sepal)")

Kode tersebut menghitung total jumlah outlier dalam DataFrame df_merged dengan langkah-langkah berikut:

  1. Akses Kolom “outlier”: df_merged[“outlier”] mengambil kolom “outlier” dalam DataFrame. Biasanya, kolom ini berisi nilai 1 untuk outlier dan 0 untuk data normal.

  2. Hitung Jumlah Outlier: .sum() menjumlahkan semua nilai dalam kolom “outlier”, sehingga menghasilkan total jumlah outlier.

  3. Cetak Hasil: print(f”Total Outlier: {total_outliers}”) menampilkan jumlah outlier yang telah dihitung.

Jika kolom “outlier” berisi nilai lain (misalnya, -1 untuk outlier dan 1 untuk normal seperti pada LOF), pastikan nilai yang dijumlahkan sesuai dengan definisi outlier dalam dataset.

Kode tersebut mengurutkan data dalam DataFrame df_merged berdasarkan kolom “distance” dari yang terkecil ke terbesar, dengan langkah-langkah berikut:
