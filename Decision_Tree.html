3. Ambil contoh threshold split (misalnya):#

  • Threshold = 52 cm

Maka kita akan membagi:

Group Left (≤ 52):

  • No: 13, 9, 6, 15, 8, 7, 4, 10 → 8 data

  • Kelas: 7 Felis chaus, 1 Felis catus

Group Right (> 52):

  • No: 12, 3, 1, 14, 2, 11 → 6 data

  • Kelas: 6 Felis catus

4. Hitung Entropy Awal (sebelum split)#

\[ Entropy(D) = -p_1 \log_2(p_1) - p_2 \log_2(p_2) \]
\[ p_1 = \frac{8}{15},\quad p_2 = \frac{7}{15} \]
\[ Entropy(D) = -\frac{8}{15} \log_2 \frac{8}{15} - \frac{7}{15} \log_2 \frac{7}{15} \approx 0.996 \]

5. Entropy setelah split#

Group Left (≤ 52):

  • 7 Felis chaus, 1 Felis catus

\[ Entropy = -\frac{7}{8} \log_2 \frac{7}{8} - \frac{1}{8} \log_2 \frac{1}{8} \approx \boxed{0.543} \]

Group Right (> 52):

  • 6 Felis catus

  • Entropy = 0

6. Information Gain:#

\[ IG = 0.996 - \left( \frac{8}{14} \cdot 0.543 + \frac{6}{14} \cdot 0 \right) \approx \boxed{0.686} \]

7. Gini Index#

Gini Awal:

\[ 1 - \left( \frac{8}{15} \right)^2 - \left( \frac{7}{15} \right)^2 \approx \boxed{0.497} \]

Gini(Left):

  • 7 chaus, 1 catus

\[ 1 - (7/8)^2 - (1/8)^2 = \boxed{0.21875} \]

Gini(Right):

  • Semua Felis catus → Gini = 0

Gini Split:

\[ \frac{8}{14} \cdot 0.21875 + \frac{6}{14} \cdot 0 = \boxed{0.125} \]

2. Threshold = 4.4 kg#

Group Left (≤ 4.4):

  • No: 13, 9, 6, 15, 8, 7, 4, 10 → 8 data

  • Kelas: 6 Felis chaus, 2 Felis catus

Group Right (> 4.4):

  • No: 2, 14, 5, 1, 12, 3, 11 → 7 data

  • Kelas: Semua Felis catus

3. Entropy:#

Entropy Awal: Tetap

\[ \boxed{0.996} \]

Entropy Left (6 chaus, 2 catus):

\[ Entropy = -0.75 \log_2 0.75 - 0.25 \log_2 0.25 \approx \boxed{0.811} \]

Entropy Right: Semua Felis catus = 0

Information Gain:

\[ IG = 0.996 - \left( \frac{8}{15} \cdot 0.811 \right) = \boxed{0.564} \]

4. Gini Index#

Gini Awal:

\[ \boxed{0.497} \]

Gini Left:

\[ Gini = 1 - (6/8)^2 - (2/8)^2 = \boxed{0.375} \]

Gini Right = 0

Gini Split:

\[ Gini_{split} = \frac{8}{15} \cdot 0.375 + \frac{7}{15} \cdot 0 = \boxed{0.2} \]

Visualisasi Decision Tree#

Ini adalah code program untuk mencari decision tree sekaligus rules

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
from sklearn.tree import DecisionTreeClassifier, plot_tree, export_text
from sklearn.preprocessing import LabelEncoder

# === Data Felis (Sesuai Tabel) ===
data = {
    "Panjang Tubuh (cm)": [55, 57, 54, 52, 56, 49, 51, 50, 48, 52, 58, 53, 47, 55, 49],
    "Berat Tubuh (kg)": [4.8, 4.6, 5.0, 4.2, 4.7, 3.9, 4.2, 4.0, 3.7, 4.4, 5.2, 4.8, 3.5, 4.6, 4.0],
    "Kelas": [
        "Felis catus", "Felis catus", "Felis catus", "Felis catus", "Felis catus",
        "Felis chaus", "Felis chaus", "Felis chaus", "Felis chaus", "Felis chaus",
        "Felis catus", "Felis catus", "Felis chaus", "Felis catus", "Felis chaus"
    ]
}
df = pd.DataFrame(data)

# === Label Encoding ===
le = LabelEncoder()
df['Label'] = le.fit_transform(df['Kelas'])  # Felis catus = 0, Felis chaus = 1 (atau sebaliknya tergantung encoding)

# === Fitur dan Target ===
X = df[["Panjang Tubuh (cm)", "Berat Tubuh (kg)"]]
y = df["Label"]

# === Train Decision Tree ===
clf = DecisionTreeClassifier(criterion="gini", max_depth=3, random_state=42)
clf.fit(X, y)

# === 1. Tampilkan Pohon Keputusan ===
print("=== Pohon Keputusan (Decision Tree) ===")
tree_rules = export_text(clf, feature_names=list(X.columns))
print(tree_rules)

=== Pohon Keputusan (Decision Tree) ===

|   |--- Panjang Tubuh (cm) <= 51.50
|   |   |--- class: 1
|   |--- Panjang Tubuh (cm) >  51.50
|   |   |--- Berat Tubuh (kg) <= 4.30
|   |   |   |--- class: 0
|   |   |--- Berat Tubuh (kg) >  4.30
|   |   |   |--- class: 1
|--- Panjang Tubuh (cm) >  52.50
|   |--- class: 0

Visualisasi Splitting#

# === Uji 2 Data Baru ===
uji_baru = pd.DataFrame({
    'Panjang Tubuh (cm)': [56, 65],
    'Berat Tubuh (kg)': [5.1, 6.2]
})

prediksi = clf.predict(uji_baru)
hasil = le.inverse_transform(prediksi)

print("\n=== Prediksi 2 Data Baru ===")
for i, (p, b) in enumerate(zip(uji_baru['Panjang Tubuh (cm)'], uji_baru['Berat Tubuh (kg)'])):
    print(f"Data {i+1}: Panjang={p}, Berat={b} → Prediksi: {hasil[i]}")

# === Visualisasi Decision Boundary ===
print("\n=== Visualisasi Decision Boundary ===")
plot_colors = "ry"
plot_step = 1.0

x_min, x_max = X['Panjang Tubuh (cm)'].min() - 10, X['Panjang Tubuh (cm)'].max() + 10
y_min, y_max = X['Berat Tubuh (kg)'].min() - 10, X['Berat Tubuh (kg)'].max() + 10

xx, yy = np.meshgrid(np.arange(x_min, x_max, plot_step),
                     np.arange(y_min, y_max, plot_step))

grid = np.c_[xx.ravel(), yy.ravel()]
grid_df = pd.DataFrame(grid, columns=['Panjang Tubuh (cm)', 'Berat Tubuh (kg)'])
Z = clf.predict(grid_df)
Z = Z.reshape(xx.shape)

plt.figure(figsize=(8, 6))
plt.contourf(xx, yy, Z, cmap=plt.cm.RdYlBu, alpha=0.6)

# Plot titik data training
for label, color in zip([0, 1], plot_colors):
    idx = np.where(y == label)
    plt.scatter(X.iloc[idx]['Panjang Tubuh (cm)'], X.iloc[idx]['Berat Tubuh (kg)'],
                c=color, edgecolor='k', label=le.inverse_transform([label])[0])

plt.xlabel("Panjang Tubuh (cm)")
plt.ylabel("Berat Tubuh (kg)")
plt.title("Decision Boundary (Panjang vs Berat)")
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()
